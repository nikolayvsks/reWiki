# Введение

Добро пожаловать на вики **Revite — приватного Minecraft-сервера с углублением в общение и творчество.** У нас вы встретите не только привычные вещи, но и множество эксклюзивов: самописные плагины, разбавляющие геймплей и система с акцентом на сплочение игроков.
