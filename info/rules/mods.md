# Разрешённые и Запрещённые модификации

{% hint style="info" %}
Используйте Ctrl+F для быстрого поиска модов!
{% endhint %}

{% hint style="info" %}
Ресурс-паки и сторонние программы, которые могут что-то делать в игре/для игры тоже относятся к модификациям.
{% endhint %}

## <mark style="color:$danger;">Запрещено</mark> использовать

{% hint style="danger" %}
Помимо перечисленных ниже модификаций, к использованию также запрещены модификации, выполняющие те же функции.

Если функция является отключаемой частью модификации — её можно отключить и использовать модификацию без этой функции.
{% endhint %}

* Чит-клиенты и чит-модификации;
* X-Ray модификации;
* Модификации, выполняющие действия за игрока; <sub><mark style="color:$info;">В т.ч. автокликеры, авторыбалка, Litematica Prinet и пр.<mark style="color:$info;"></sub>
* SeedCracker и другие модификации для вычисления сида мира;
* Авто-тотем;
* Отображение мобов и пещер в картах и миникартах;
* Bedrock Miner;
* Моды, отображающие содержимое шалкера когда он не открыт и не в инвентаре; <sub><mark style="color:$info;">Например, Shulker box hints в моде Peek<mark style="color:$info;"></sub>
* Tweakeroo;
* Обводка блоков руды для упрощения поиска;
* Freecam с возможностью пролетать через блоки;

## <mark style="color:$success;">Разрешено</mark> использовать

{% hint style="success" %}
Помимо перечисленных ниже модификаций, к использованию также разрешены модификации, выполняющие те же функции.
{% endhint %}

* Модификации, вносящие исключительно внутренние изменения и **никак** не влияющие на геймплей (оптимизация, библиотеки и пр.);
* Модификации, вносящие косметические изменения, которые **никак** не затрагивают баланс игры;
* PlasmoVoice;
* EmoteCraft;
* Litematica; <sub><mark style="color:$info;">Easy Place Mode также разрешён<mark style="color:$info;"></sub>
* LocatorHeads;

* Better Mount HUD;
* GoogleChat;
* Iris Shaders;
* Gamma (Яркость);
* LambDynamicLights;
* Paginated Advancements & Custom Frames;
* Sodium, Sodium Extra и Resee's Sodium Options;
* Sound Physics Remastered;
* Symbol Chat;
* uku's Armor HUD;
* Zoomify;
* AppleSkin;
* BetterF3;
* Stendhal;
* Essential;
