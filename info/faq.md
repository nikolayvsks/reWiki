# Часто задаваемые вопросы

{% hint style="info" %}
Статья будет дополняться новыми вопросами по мере их появления.
{% endhint %}

### Какая версия сервера?

На сервер можно зайти с версии `1.21.7` или `1.21.8` (поддерживаются обе версии).

Сервер будет обновляться на новые версии по мере их готовности к комфортной игре.

### Можно ли зайти с телефона/приставки/консоли?

Нет. Сервер поддерживает только Java Edition, которая официально доступна только для компьютеров.

### Когда посмотрят мою заявку?

Заявки рассматриваются в течение 12 часов.

Во-первых, модераторы тоже люди и могут быть заняты, в том числе и личными делами на сервере.

Во-вторых, мы придерживаемся некоторых правил при принятии заявки, чтобы избежать дальнейших проблем.

Именно поэтому мы не всегда можем принимать заявки так быстро, как многим хотелось бы. Пожалуйста, будьте терпеливыми.

### Необходима ли лицензия?

Использовать лицензию необязательно — сервер доступен с пиратских аккаунтов.

Но используя лицензию вы получите лицензионную авторизацию на сервер без паролей.

### Какая валюта сервера?

Глубинносланцевая и обычная алмазная руда.

### Когда открылся сервер?

Сам сервер Revite существует с октября 2022, хоть и его корни идут ещё раньше этой даты.

Нынешний, четвёртый сезон сервера открылся 4 октября 2025.

### Когда закончится сезон?

Точного овтета дать нельзя, ведь сезон длится в зависимости от интереса игроков. Пока сезон интересен игрокам — он будет жить. Тем не менее, четвёртый сезон продлится не менее 4-х месяцев.

### Есть ли у сервера модпак?

У сервера есть [официальный модпак](https://modrinth.com/modpack/revite_modpack) с самыми необходимыми для игры модами. Его можно использовать в качестве основы и наполнять своими модами.
